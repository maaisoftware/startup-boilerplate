# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Short version

Be respectful. Assume good intent. Disagree without being rude. No harassment, no discrimination, no personal attacks.

## Enforcement

Report violations privately by opening a [GitHub security advisory](https://github.com/maaisoftware/startup-boilerplate/security/advisories/new) (we use this for all confidential reports, not just security). The maintainers investigate and respond within 5 business days.

Consequences for violations may include warnings, temporary bans, or permanent bans from the project, at the maintainers' discretion.

## Scope

The Code of Conduct applies within all project spaces — issues, pull requests, discussions, commit messages, and any external spaces where someone represents this project.
